## 项目简介

CSC8507-08 是一个基于 C++ 和 OpenGL 的 3D 游戏项目。

### 技术亮点

- **ECS 架构** — 自研 Entity-Component-System 框架
- **ImGui UI 系统** — 完整的菜单、HUD、物品栏、通知系统
- **多人联网** — 基于 ENet 的客户端-服务端架构
- **导航系统** — NavMesh 寻路
- **物理系统** — 碰撞检测与响应

### 我的贡献

- UI 系统设计与实现（全屏菜单、设置面板、动作通知等）
- 光标管理系统重构
- 无边框全屏模式
- 调试模式（DevMode）架构
